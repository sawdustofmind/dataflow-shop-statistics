# dataflow-shop-statistics
DataFlow is the app for calculation statistics for shops sales data
## Launching
### App
`go run cmd/api/main.go `
### Test
`go test ./...`
### Bench
`go test -test.bench=. ./...`

## Architect
Now this is one API server. 

All the controllers code is generated by swagger.

Storage is in-memory and it's using tree-map to manage fast sum calculation
